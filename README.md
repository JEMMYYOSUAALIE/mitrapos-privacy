# MitraPOS Privacy Policy

Repository publik GitHub Pages ini hanya berisi kebijakan privasi MitraPOS.

Target URL: `https://jemmyyosuaalie.github.io/mitrapos-privacy/`

## Publikasi

1. Cabut GitHub PAT lama yang sudah terekspos.
2. Buat repository publik baru bernama `mitrapos-privacy` pada akun `jemmyyosuaalie`.
3. Upload hanya `index.html` dan `README.md` dari folder ini ke branch `main` repository tersebut.
4. Buka **Settings > Pages**, pilih **Deploy from a branch**, branch `main`, folder `/ (root)`.
5. Tunggu deployment selesai, lalu pastikan target URL menghasilkan HTTP 200 tanpa login.

Jangan menyalin source aplikasi, keystore, `keystore.properties`, backup aplikasi, atau token ke repository privacy policy.
