Kebijakan Privasi MitraPOS

Terakhir diperbarui: 1 Juli 2026

Kebijakan ini menjelaskan cara aplikasi MitraPOS mengakses dan menangani data. Pertanyaan privasi dapat dikirim ke jemmyalie9@gmail.com.

Data yang diproses

MitraPOS memproses data yang dimasukkan pengguna untuk menjalankan fungsi kasir, termasuk profil toko, produk dan gambar produk, kategori, stok, pelanggan, transaksi, meja, lantai, session kasir, laporan, metode pembayaran, dan pengaturan printer. Data pelanggan dapat mencakup nama, nomor telepon, email, alamat, dan catatan apabila pengguna memilih untuk memasukkannya.

Aplikasi juga memproses data employee lokal berupa nama, email opsional, role, hak akses, status akun, histori login, serta audit aktivitas. PIN dan recovery code tidak disimpan sebagai teks asli; aplikasi menyimpan salted cryptographic hash pada perangkat.

Penyimpanan, pengumpulan, dan pembagian

Data operasional, akun employee, session, audit log, PIN hash, dan recovery hash disimpan secara lokal pada perangkat. Pengembang MitraPOS tidak mengumpulkan, mengirim, menjual, atau membagikan data tersebut ke server pengembang maupun pihak ketiga.

Ekspor laporan, pembuatan backup, restore, atau pembagian file hanya dilakukan atas tindakan pengguna. Pengguna menentukan sendiri aplikasi atau lokasi tujuan pembagian. Setelah file dibagikan atau disimpan di luar MitraPOS, file tersebut berada di bawah kendali pengguna dan penyedia tujuan yang dipilih.

Akses perangkat

Akses foto digunakan hanya ketika pengguna memilih gambar produk. Izin Bluetooth atau Perangkat di sekitar digunakan untuk menemukan dan menghubungkan printer termal. Pada Android 11 dan versi lebih lama, sistem Android dapat memerlukan izin lokasi untuk pemindaian Bluetooth; MitraPOS tidak menggunakan izin tersebut untuk melacak atau mengumpulkan lokasi.

Keamanan

PIN numerik diproses menggunakan salted PBKDF2 hash dan tidak dikirim ke server. Aplikasi menyediakan lockout setelah percobaan PIN gagal dan dapat mengunci sesi login setelah tidak aktif. Karena seluruh data disimpan lokal, pengguna tetap bertanggung jawab menjaga keamanan perangkat, recovery code, dan file backup. Simpan backup hanya di lokasi tepercaya.

Retensi dan penghapusan

Data dipertahankan di perangkat sampai pengguna mengubah atau menghapusnya, mengarsipkan data, memilih fitur Hapus Semua Data Lokal, menghapus data aplikasi melalui pengaturan Android, atau menghapus instalasi aplikasi. Owner dapat menghapus employee yang belum pernah digunakan atau mengarsipkan employee yang memiliki histori.

File backup atau ekspor yang telah dibuat harus dihapus secara terpisah dari lokasi penyimpanannya. MitraPOS tidak memiliki akses untuk menghapus salinan file yang sudah dipindahkan atau dibagikan oleh pengguna.

Akun offline

Owner dan employee adalah identitas lokal yang hanya berlaku pada satu instalasi perangkat. MitraPOS tidak membuat akun online, tidak menyediakan sinkronisasi lintas perangkat, dan tidak mengoperasikan layanan akun pada server pengembang.

Anak-anak

MitraPOS merupakan alat operasional bisnis untuk pengguna dewasa dan tidak ditujukan untuk anak-anak.

Perubahan kebijakan

Kebijakan ini dapat diperbarui ketika fungsi atau praktik data aplikasi berubah. Tanggal pembaruan terbaru akan ditampilkan pada halaman ini.
